# 🏛️ SSVD Constructions
> **PRESTIGE. PERMANENCE. PRECISION.**

Welcome to the digital residence of **SSVD Constructions**—a premium portfolio website built for Visakhapatnam's premier custom estate builder and structural contractor. This is not a typical construction corporate template. It is designed to feel like an editorial journal, mirroring the visual restraint, spacious grids, and prestigious branding of elite international architecture and design studios.

---

## 🎨 Visual Identity & Design System

The platform's aesthetics reflect physical longevity, raw materiality, and legacy:

*   **Warm Ivory (`#F7F4EF`)**: Emits warmth, editorial paper quality, and elite architectural layouts.
*   **Rich Charcoal (`#1A1A1A`)**: A deep, low-contrast black providing premium structure and legibility.
*   **Brass Gold (`#B08D57`)**: Used sparingly for highlight accents, reminiscent of finished metal fittings.
*   **Burnt Terracotta (`#B8643C`)**: Representing raw clay, masonry, and foundational bricks.
*   **Typography**:
    *   *Editorial Headings*: **Cormorant Garamond** (Serif) — for large-scale statements.
    *   *Body Copy*: **Plus Jakarta Sans** (Sans-serif) — for highly legible, warm descriptions.
    *   *Technical Details & UI*: **Outfit** (Geometric Sans) — for coordinates, badges, inputs, and numbers, reinforcing mathematical precision.

## 🛠️ High-End Interactive Features

### 1. Entrance Preloader
A full-screen luxury cover featuring custom branding and a gold line loading animation. Designed to mask raw asset loads and smooth the initial transition.

### 2. Autoplay Voice Welcome
When a visitor performs their first micro-interaction on the page (scrolling, clicking, or dragging), a natural human-voiced welcoming narration plays invisibly: *"Welcome to SSVD Constructions."*

### 3. Interactive "Blueprint to Reality" Comparison Slider
An interactive slider section showcasing the **Rushikonda Cliff Villa**. Users can click and drag a brass divider to dynamically slide open a detailed technical vector blueprint sketch (`assets/villa-blueprint.png`) overlaid directly on top of the final physical rendering (`assets/proj-villa.png`), highlighting structural alignment and execution.

### 4. Background Architectural Drafting Grid
Faint, thin layout lines are dynamically drawn vertically down the page canvas, mimicking the look and precision of an architect's drafting board.

### 5. Dynamic Background Services Grid
Hovering over services like *Residential Construction*, *Commercial Development*, or *Structural Engineering* transitions the border layout and dynamically blends the corresponding 8K photography into the grid background.

## ⚙️ Quick Start & Development

The project is built on vanilla web technologies powered by **Vite** for zero-dependency development.

### 1. Install Dependencies
```bash
npm install
```

### 2. Launch Development Server
```bash
npm run dev
```

### 3. Build Production Bundle
```bash
npm run build
```
